# HR-Payroll-Management
## Project Introduction
This project introduces students to how organizations manage employee payroll using excel.
Students will work with employee data to calculate salaries, allowances, deductions, and net pay.
### Skills You Will Obtain After Completing Thos Project
*** After working on this project, students will be abale to:***
1. Organize employee and payroll data in excel
2. Use arithmetic and logic Excel formulas
3. Calculate gross salary, deductions, amd net pay
4. Apply condtional functions such as IF and SUMIF
5. Create payroll summaries for management
6. Identify payroll errors using excel logic
